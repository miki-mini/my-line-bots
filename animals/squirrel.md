# 🐿️ リスのほっぺたどんぐりゲーム

MediaPipe Hands を使用した、2人対戦型の手を使ったインタラクティブゲームです。
画面上から落ちてくるどんぐりを手でキャッチして、リスのほっぺたをいっぱいに膨らませた方が勝ちです！

## 機能
- **手認識**: カメラで手を認識し、画面上のアイテムをキャッチできます。
- **2人対戦**: 画面の左半分がPlayer 1、右半分がPlayer 2のエリアになります。
- **表情変化**: スコアに応じてリスのほっぺたがリアルに膨らんでいきます。
- **アイテム**:
    - 🌰 **どんぐり**: +10点
    - 🥔 **くるみ**: +30点
    - ⚠️ **イガ栗**: -50点（ダメージ顔になります）
- **難易度選択**: Normal / Hard (スピードアップ＆イガ栗増加)
- **Fever Time**: 残り10秒でどんぐりが大量に降り注ぎます！
- **BGM**: "Hurry Up" - 焦燥感を煽る楽しいBGM

## 使用技術
- MediaPipe Hands
- HTML5 Canvas
- Web Audio API (効果音)
